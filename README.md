# Rapidly_Exploring_Random_Tree

## Maintainer   
- [Hanqing Zhang], <<hzhang8@wpi.edu>>, WPI    

## Dependencies

- ROS Kinetic   
- Ubuntu 16.04   
- C++11 standard library   

## Compile and Run
- Switch to the right path   
- g++ -std=c++11 X.h X.cpp -o X   
- ./X
- python display.py
